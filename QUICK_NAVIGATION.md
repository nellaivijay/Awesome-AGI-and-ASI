## 🔧 Practical Guides

### Setting Up Local AI
1. Install [Ollama](infrastructure/llm-frameworks.md)
2. Download Gemma 4 or similar model
3. Configure local inference
4. See: [LLM Application Frameworks](infrastructure/llm-frameworks.md)

### Building Your First AI Agent
1. Choose framework: [LangGraph](infrastructure/llm-frameworks.md) or [CrewAI](build/frameworks.md)
2. Define agent role and tools
3. Implement memory and context
4. Test and iterate
5. See: [Agents](build/agents.md)

### Implementing RAG
1. Select vector database: [RAG and Vector Databases](infrastructure/rag-vector-db.md)
2. Chunk and embed documents
3. Set up retrieval pipeline
4. Add LLM for generation
5. See: [RAG and Vector Databases](infrastructure/rag-vector-db.md)

### Fine-Tuning Models
1. Choose approach: [LoRA](infrastructure/fine-tuning.md) or full fine-tuning
2. Prepare dataset
3. Configure training pipeline
4. Evaluate and iterate
5. See: [LLM Fine-Tuning Techniques](infrastructure/fine-tuning.md)
