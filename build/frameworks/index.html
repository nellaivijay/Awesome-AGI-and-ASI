    Frameworks and Platforms¶

    The infrastructure powering Large Agentic Models (LAMs) and multi-agent systems. The conversation has moved past simple LLMs and chatbots -- AGI is about systems that take actions. These frameworks let you build, orchestrate, and deploy AI agents that can reason, browse, code, and collaborate autonomously.

    Agent Communication Protocols¶

    Standardized protocols enabling agents to discover tools, communicate with services, and coordinate with each other -- the TCP/IP of the agentic era.

    Protocol Description Links
    MCP (Model Context Protocol) Anthropic's protocol for connecting AI to external tools/data. Adopted by Claude, Cursor, Windsurf. Emerging agent-tool standard. modelcontextprotocol.io, GitHub
    A2A (Agent-to-Agent Protocol) Google's protocol for inter-agent communication. Discover capabilities, negotiate tasks, collaborate securely. google.github.io/A2A
    ACP (Agent Communication Protocol) BeeAI's protocol for structured agent messaging. Streaming, async tasks, multi-turn interactions. github.com/i-am-bee/beeai
